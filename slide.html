AceSlide 使用 HTML 的演示文档工具

邹业盛 / yeshengzou@gmail.com / 2012-03-28

  • AceSlide 以 Frame 为单位来组织演示文档。
  • 一个 Frame 即是一个 div ，加上一个 class="frame"。
  • 最关键的，你所做的一切都是在写 HTML ，随意。

要使用 AceSlide 需要做的事包括：

  1. 引入一堆 js 文件：dojo.js, main.js, frame.js, init.js
  2. 引入两个 css 文件：reset.css, main.css
  3. 开始写第一个 frame 。
  4. 在最后引入一段 div[class="toolbar"]。
  5. 在最后引入一段 div[class="toc"]。
  6. 在最后补一个 script 标签，否则 chrome 下布局有问题。

下一页展示了最简单的完整代码。

AceSlide 中的内容只是普通的 HTML 文本，所以你可以随意地使用 CSS 来调整样式。

比如对于第一页标题页，我通常会添加一段 CSS ：

<style type="text/css"> #title_page h1{ height: 100px; line-height: 100px; margin-top: 200px; text-shadow: 2px 2px 2px #FFF; font-size: 40px; letter-spacing: 5px; text-align: center; background-color: #FFD600; } #index_page li{ font-size: 28px; } </style>

当然，对于制作演示文档，对于你需要的格式，直接在元素中写 style 是不错的选择。

<p style="width: 500px; margin: 15px auto;"><i>顺序，即时结果，等待，阻塞，浪费时间</i></p>

Frame 中的节点是可以按顺序依次出现的……

只需要在父级节点当中，添加一个 class="overlay"，如：

<div class="frame overlay" name="顺序出现"> <p>就像本页的效果一样……</p> <p> Frame 中的节点是可以按顺序依次出现的……</p> <p>只需要在父级节点当中，添加一个 <i>class="overlay"</i>，如： </p> <ul class="overlay"> <li>就如同代码显示……</li> <li>嵌套使用 <i>overlay</i> …… </li> <li>也是可以的。</li> </ul> </div>
  • 就如同代码显示……
  • 嵌套使用 overlay ……
  • 也是可以的。
  • 对于overlay是可以在节点当中，使用 tabindex 指定出现的顺序的。
  • 使用class="phantom"可以提前占位。
<div class="frame overlay" name="指定顺序"> <ul class="overlay phantom" tabindex="1-"> <li tabindex="2-">对于<i>overlay</i>是可以在节点当中， 使用 <i>tabindex</i> 指定出现的顺序的。</li> <li class="phantom" tabindex="3-">使用<i>class="phantom"</i>可以提前占位。</li> </ul> </div>
  • tabindex="1-"表示的是“从第1帧开始”。
  • 而tabindex="1"表示仅第1帧（从第2帧开始就不可见）。
  • tabindex计数，对于当前 frame 是一个全局量，不按节点分层次关系。
  • 源码中的 server.py 是一个使用 tornado 实现的控制服务器。
  • 如果你安装了 tornado ，可以直接把这个 server.py 启动起来。
    • python server.py --port=8888
  • 注意在 server.py 源码中直接配置一个密码：
    • PASSWORD = '123'
  • 点击右下角的“控制”，作为控制方（需要密码）连接到控制服务器。
  • 点击右下角的“连接”，作为被控制方连接到控制服务器。
  • 控制方翻页时，被控制方会同步翻页。
  • 直接访问控制服务器的地址，如 http://localhost:8888/ ，输入密码后，作为控制方，进入一个专门的控制平台，它的翻页操作同样影响所有的被控制方（你可以使用手机访问来控制）。
